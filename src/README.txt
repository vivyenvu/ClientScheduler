Client Appointment Scheduling System

This system is designed to allow certain users to add, update, and delete Customers and Appointments. The GUI includes different schedule displays and report displays.

Author: Vivyen Vu
Contact: vvu11@wgu.edu
Application version: 1.8
Date: 9/3/22

IDE: IntelliJ IDEA Community 2021.1.3
JDK: Java SE 17.0.1
JavaFX: JavaFX-SDK-11.0.2

How to use program:
Launch application from same IDE listed above.
You can login with username: "user" and password: "user" or with the credentials username: "admin" and password "admin".
On the Main Menu, you can add, update, or delete Customers and Appointments.
To view different time ranges for your Appointments, you can select different radio buttons.
There is an additional Reports button that will allow you to view three different reports after clicking the associated buttons.

Additional report:
This additional report allows the user to view a count of how many customers are from each country.

MySQL Connector driver version: mysql-connector-java-8.0.29
